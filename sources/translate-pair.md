Шаг 1. Изучи данный материал прежде чем ответить:
- Определи насколько сложный текст, как много ошибок, что нужно будет изменить.
- Формируй без скриптов за один проход.

Шаг 2. Трансформируй в русский B2:
- Прямой порядок внутри частей: подлежащее → сказуемое → дополнение (например: `Пулями летают лифты` → `Лифты быстро движутся`).
- Допустимы сложные предложения с 1 простой придаточной | второй частью (например: `Я открыл дверь, когда услышал звонок.`).
- Избегай 2+ вложений (например: НЕТ `Рынок работает, где торговцы, которые продают товары, привезённые издалека.`).
- Разворачивай причастные/деепричастные через "который/где/когда" (например: `наводнённый торговцами рынок` → `рынок, где много торговцев`).
- Глаголы/прилагательные → популярные B2-эквиваленты (например: `суетится` → `работает`, `осуществляет движение` → `движется`).
- Существительные (предметы, люди, места) → сохраняй точно для смысла.
- Метафоры и образные выражения → прямые описания (например: `пулями летают` → `быстро движутся`).
- Избегай канцелярита и эвфемизмов.
- Сохраняй: факты, имена, числа, причинно-следственные связи, эмоциональную окраску.
- Пропусти заголовки, ссылки, метаданные, разметку, смайлики, временные метки.
- Исправь опечатки, склейки слов, случайные символы и цифры внутри слов.

Шаг 3. Переведи русский на английский:
- Переводи готовый русский текст точно по смыслу.
- Используй простую лексику B2.
- Выбирай формулировки проще, глаголы разговорные, делай приоритет к фразовым глаголам , спользующимся в разговорном.
