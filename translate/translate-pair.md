# Шаг 1. - **Изучи данный материал.**:

- [ ] Определи насколько сложный текст, как много ошибок, что нужно будет изменить.

# Шаг 2. - **Отформатируй в русский B2.**:

- [ ] Используй прямой порядок слов внутри каждой части: подлежащее → сказуемое → дополнение.
- [ ] Разрежь исходный текст строго по 4 предложения, игнорируя абзацы и форматирование оригинала.
- [ ] Разбей на отдельные предложения при >2 вложений (например: "Он знал, что мать, которая работала, пока он спал, устала" → "Мать работала, пока он спал. Он знал, что она устала").
- [ ] Разверни причастные | деепричастные через "который | где | когда" (например: "часть, давно похороненная" → "часть, которая давно спала").
- [ ] Избегай архаизмов и сложных слов.
- [ ] Избегай устойчивых выражений с неочевидным значением.
- [ ] Замени метафоры на прямые описания.
- [ ] Замени глаголы | прилагательные на популярные B2-эквиваленты.
- [ ] Сохрани существительные (предметы, люди, места) точно для смысла.
- [ ] Сохрани текст в "кавычках".
- [ ] Используй прямой порядок слов автора (например: "Мне нужно идти," Мама Джонс сказала).

# Шаг 3. - **Переведи русский на английский.**:

- [ ] Переведи готовый русский текст точно по смыслу.
- [ ] Используй простую лексику B2.
- [ ] Выбирай формулировки проще и глаголы разговорные, делай приоритет к фразовым глаголам, использующимся в разговорном.

# Шаг 4. - **Повтори Шаг 2 + Шаг 3 для других пар пока не закончится текст.**:

- [ ] Обработай весь предоставленный сырой текст за один раз.
- [ ] Соблюдай количество разделителей между парами 1 разделитель, внутри пары - 3 разделителя.

# Шаг 5. - **Запиши результат в файл.**:
